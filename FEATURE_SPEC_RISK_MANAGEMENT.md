# 新功能规格书：风险管理与仓位建议功能 (Risk Management & Position Sizing)

## 🎯 功能概述

在单股详情页面新增 **【风险管理】** 面板，为用户提供：
1. **止损价建议** - 基于历史支撑位自动计算
2. **止盈价建议** - 基于 ABC 波段理论预测
3. **仓位大小建议** - 基于波幅 (AMP) 的动态仓位
4. **风险/收益比显示** - 实时计算和展示
5. **风险评级与交易推荐** - 综合的交易可行性评估

---

## 🔧 核心功能模块

### A. 止损价建议 (基于历史支撑位)

支撑位计算方法:
- **近期支撑** (20日最低): 过去20个交易日的最低价
- **中期支撑** (50日最低): 过去50个交易日的最低价  
- **长期支撑** (100日最低): 过去100个交易日的最低价
- **技术支撑**: 前两个低点的平均值

止损级别选项:
- 超激进 (-2.0%) - 小虧即出
- 激进 (-3.0%) - 推薦選項  
- 适中 (-4.0%) - 平衡型
- 保守 (-5.0%) - 风险承受力强
- 超保守 (-6.0%) - 最大虧損容忍度

---

### B. 止盈价建议 (基于 ABC 波段理论)

基于 Fibonacci 比例计算:
- **保守目标** (38.2% 反弹): 輕微反彈
- **中等目标** (61.8% 反弹): 標準反彈 ⭐推薦
- **激进目标** (100% 反弹): 完全反轉

---

### C. 仓位大小建议 (基于波幅 AMP)

仓位计算公式:
```
仓位 = (总资金 × 风险百分比 × 风险倍数) / (买入价 × 止损百分比)

风险倍数:
- 高波幅 (AMP > 平均 × 1.3): 0.6 (减少仓位)
- 中波幅: 1.0 (正常)
- 低波幅 (AMP < 平均 × 0.9): 1.2 (增加仓位)

风险百分比按风险偏好:
- 保守: 2%
- 中等: 4%  
- 激进: 6%
```

---

### D. 风险/收益比分析

关键指标:
- 风险金额 = 买入价 - 止损价
- 收益金额 = 止盈价 - 买入价
- 风险/收益比 = 收益金额 / 风险金额
- 期望值 = 胜率 × 平均收益 - (1-胜率) × 平均亏损

---

### E. 综合风险评级

评分体系 (总分100):
- 波幅评分 (0-20): 基于当前AMP vs 历史平均
- 支撑位评分 (0-20): 基于支撑位强度
- 成交量评分 (0-20): 基于成交量趋势
- 风险/收益评分 (0-20): 基于比例
- 胜率评分 (0-20): 基于历史胜率

风险等级:
- 🟢 超低风险 (85-100): 可按计划进行
- 🟡 低风险 (70-85): 可交易但监控
- 🟡 中等风险 (50-70): 需谨慎
- 🔴 高风险 (30-50): 不推荐
- 🔴 超高风险 (0-30): 严禁交易

## 🛠️ 主要类设计

### RiskManagementEngine 类

```
RiskManagementEngine
├── support_analysis: SupportLevelAnalysis
├── profit_analysis: ProfitTargetAnalysis  
├── position_sizing: PositionSizingEngine
├── risk_reward: RiskRewardAnalysis
└── comprehensive_assessment: ComprehensiveRiskAssessment

主要方法:
- generate_full_report(): 生成完整报告
- calculate_support_levels(): 计算支撑位
- calculate_profit_targets(): 计算止盈目标
- calculate_position_size(): 计算仓位
- assess_risk(): 评估风险
```

## 📐 UI 界面位置

在单股详情页面添加 **【🛡️ 风险管理】** 面板，包含5个子模块:

1. **买入点建议** - 理想进场价格和激进进场价格
2. **止损/止盈设置** - 可选的止损级别和止盈目标
3. **仓位大小建议** - 按风险偏好的仓位分配
4. **风险/收益比分析** - 详细的比例和期望值计算
5. **综合风险评级** - 风险评分和综合建议
